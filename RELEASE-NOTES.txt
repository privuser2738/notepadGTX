INSTALLATION METHODS
--------------------

Method 1: Complete Installer (Recommended for end-users)
  $ tar -xzf notepadGTX-installer.tar.gz
  $ cd notepadGTX-installer
  $ sudo ./install.sh

Method 2: Run CLI Binary Directly
  $ chmod +x notepadGTX-cli
  $ ./notepadGTX-cli -f myfile.txt

Method 3: Run GUI AppImage Directly
  $ chmod +x notepadGTX-1.0.0.AppImage
  $ ./notepadGTX-1.0.0.AppImage

Method 4: Install Debian Package
  $ sudo dpkg -i notepadgtx_1.0.0_amd64.deb

================================================================================

USAGE
-----

After installation:

Auto-detect mode:
  $ notepadgtx

Force CLI mode:
  $ notepadgtx --cli
  $ notepadgtx -c

Force GUI mode:
  $ notepadgtx --gui
  $ notepadgtx -g

Open a file:
  $ notepadgtx myfile.txt
  $ notepadgtx --cli -f myfile.txt

KEYBOARD SHORTCUTS
------------------

Both CLI and GUI modes:
  Ctrl+U - Save file
  Ctrl+N - Save As (new filename)
  Ctrl+V - Open file
  Ctrl+K - Exit application

Additional in CLI mode:
  Arrow Keys - Navigate
  Home/End - Beginning/end of line
  Enter - New line
  Backspace/Delete - Remove characters

TECHNICAL DETAILS
-----------------

CLI Implementation:
  - Built with Node.js built-in modules (readline, fs)
  - Uses ANSI escape codes for terminal control
  - No external dependencies
  - Packaged with pkg (Node.js → native binary)
  - Size: 45MB (includes Node.js runtime)

GUI Implementation:
  - Built with Electron framework
  - HTML/CSS/JavaScript interface
  - Modern dark theme
  - Packaged with electron-builder
  - Size: 100MB (includes Chromium runtime)

Binary Compatibility:
  - Target: Linux x64 (64-bit)
  - No external dependencies required
  - Fully standalone executables
